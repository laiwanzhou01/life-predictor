# 程序员寿命预测器

基于《程序员延寿指南》的科学研究数据开发的交互式寿命预测工具。

## 🎯 项目简介

这是一个基于大量学术研究数据的寿命预测网站，参考了GitHub上的开源项目[《程序员延寿指南》](https://github.com/geekan/HowToLiveLonger)中的科学研究成果。

该工具通过用户的生活习惯数据，计算全因死亡率(ACM - All-Cause Mortality)的变化，并预测对预期寿命的影响。

## ✨ 主要功能

- 📊 **寿命预测**: 基于40+个生活习惯因素进行综合计算
- 📈 **可视化展示**: 
  - 寿命对比横向柱状图
  - 类别影响分析图表
  - 动态彩色进度条
- 🎯 **类别分析**: 
  - 7大类别综合影响统计
  - 详细因子分解说明
  - 智能优先级排序
- 💡 **个性化建议**: 根据用户情况给出优先级排序的改善建议
- 🎨 **现代化UI**: 响应式设计，支持桌面和移动设备
- 📚 **科学依据**: 所有数据均来自学术论文和大规模研究
- ⚖️ **V2.3中等规则**: 平衡激励性与现实性，寿命上限70-108岁

## 📋 评估因素（40+项）

### 饮食习惯
- **固体**：肉类、蔬果、辣椒、坚果、加工食品、槟榔、碳水比例、蛋白质来源
- **液体**：咖啡、牛奶、茶、含糖饮料、酒精

### 生活习惯
- **气体与光照**：吸烟状况、晒太阳
- **药物与补充剂**：二甲双胍、复合维生素、葡萄糖胺、亚精胺
- **运动**：挥拍运动、剧烈运动、走路、做家务（老年男性）
- **日常习惯**：刷牙、泡澡

### 睡眠与久坐
- 睡眠时长（最优7小时）
- 入睡时间（最优22:00-24:00）
- 久坐时长

### 心理与体重
- 情绪状态（乐观/悲观）
- 体重状况（肥胖/超重/正常）

## 🧮 计算方法

### ACM到寿命的转换
根据指南中的公式：

```
ΔLifeSpan = (1/(1+ΔACM) - 1) × 10
```

其中：
- `ΔACM`: 全因死亡率的变化值（百分比）
- `ΔLifeSpan`: 预期寿命的变化（年）

### 综合评估
各个生活习惯因素对ACM的影响进行累加（简化模型），然后转换为寿命变化。

**注意**: 实际上各因素之间可能存在交互作用，本工具使用的是简化的累加模型。

### 寿命极限与规则版本

**当前版本：V2.3 中等强度规则** ⭐推荐

系统设置了以下限制：
- **最小寿命**: 不低于当前年龄
- **动态上限**: 根据ACM值分8档（70-108岁）
  - 极度健康（ACM < -200%）: 108岁
  - 非常健康（ACM -150~-200%）: 100岁
  - 健康（ACM -100~-150%）: 93岁
  - 其他档次: 70-90岁
- **寿命变化上限**: +32年/-35年
- **收益递减**: ACM<-120%时触发
- **年龄调整**: 60岁以上每年递减0.15岁

**特点**：
- ✅ 平衡激励性与现实性
- ✅ 不会突破人类极限
- ✅ 对中等健康影响小
- ✅ 基于科学数据优化

## ⚠️ 重要声明

1. **仅供参考**: 本工具的预测结果仅供参考，不构成医疗建议
2. **相关性非因果**: 所有数据显示的是统计相关性，并非严格的因果关系
3. **个体差异**: 每个人的健康状况不同，实际情况可能与预测存在偏差
4. **专业咨询**: 如有健康问题，请咨询专业医生
