# 严格规则说明 V2.1

## 🎯 为什么需要严格规则？

原始版本的寿命预测可能过于乐观，因为：
1. 简单的ACM累加忽略了因素间的相互作用
2. 没有考虑收益递减效应
3. 对关键风险因素的惩罚不够
4. 寿命上限设置不够现实

**V2.1版本**采用基于科学研究的严格规则，让预测更贴近现实。

---

## 📊 严格规则详解

### 规则1: 寿命上限分级限制

**旧规则**：
- 固定上限：115岁

**新规则**（基于ACM值动态调整）：

| ACM范围 | 寿命上限 | 说明 |
|---------|---------|------|
| < -200% | 105岁 | 极度健康（0.01%人群） |
| -150% ~ -200% | 95岁 | 非常健康（1%人群） |
| -100% ~ -150% | 90岁 | 健康（10%人群） |
| > -100% | 85岁 | 普通或不健康 |

**科学依据**：
- 目前有可靠记录的最长寿者：122岁（Jeanne Calment）
- 超级百岁老人（110+）：全球约300-450人
- 百岁老人（100+）：仅占人口0.02%
- 即使极度健康，生物学上限也存在

---

### 规则2: 负面因素惩罚加重

**旧规则**：
```
总ACM = 各因素ACM相加
```

**新规则**（非线性惩罚）：
```
基础ACM = 各因素ACM相加
如果负面ACM > 50%:
    额外惩罚 = (负面ACM - 50) × 0.2
    总ACM += 额外惩罚
```

**示例**：
```
负面因素总计：+80% ACM

旧规则：直接计算，影响固定
新规则：+80% + (80-50)×0.2 = +86% ACM
额外惩罚：+6%
```

**科学依据**：
- 多个风险因素会产生协同效应
- 研究显示：吸烟+饮酒的危害 > 两者相加
- 不健康习惯越多，相互作用越强

---

### 规则3: 积极因素收益递减

**旧规则**：
```
积极因素无上限，可以无限累加
```

**新规则**（边际效应递减）：
```
如果积极ACM < -150%:
    超出部分 = 积极ACM + 150
    递减效果 = 超出部分 × 0.6
    总ACM = 总ACM - 超出部分 + 递减效果
```

**示例**：
```
积极因素总计：-200% ACM

旧规则：-200% 完整计入
新规则：-150% + (-50% × 0.6) = -180% ACM
收益递减：20% 效果损失
```

**科学依据**：
- 边际效益递减规律
- 研究显示：运动时间从0→30分钟收益大，但从60→90分钟收益小
- 人体生理极限存在，不能无限优化

---

### 规则4: 关键风险因素额外权重

**识别的关键风险**：
1. 重度吸烟（≥10支/天）
2. 大量饮酒（>200g纯酒精/周）
3. 肥胖（BMI ≥ 30）
4. 经常嚼槟榔

**旧规则**：
```
按标准ACM值计算
```

**新规则**：
```
关键风险总ACM × 1.3（额外30%惩罚）
```

**示例**：
```
重度吸烟：+54% ACM
大量饮酒：+50% ACM

旧规则：+104% 总计
新规则：+104% + (104% × 0.3) = +135% ACM
额外惩罚：+31%
```

**科学依据**：
- WHO确认的一级致癌物（烟草、酒精、槟榔）
- 这些因素会显著缩短寿命，且难以逆转
- 研究显示其影响被普遍低估

## 🔬 科学依据

### 寿命上限研究

1. **Jeanne Calment案例** (122岁)
   - 唯一经过严格验证的超级长寿案例
   - 争议：有研究质疑其真实性

2. **超级百岁老人研究**
   - 全球110岁以上：300-450人
   - 115岁以上：仅20-30人
   - 120岁以上：仅1人有记录

3. **生物学极限理论**
   - Hayflick极限：细胞分裂次数有限
   - 端粒缩短：随年龄递减
   - 多数科学家认为人类极限在120-125岁

### 协同效应研究

1. **Framingham心脏研究**
   - 多风险因素呈乘法关系，非加法
   - 吸烟+高血压+高胆固醇 ≠ 三者简单相加

2. **护士健康研究**
   - 5个以上不良习惯：死亡风险增加3-5倍
   - 非线性增长曲线

### 边际效益研究

1. **运动收益曲线**
   - 0→30分钟/天：大幅降低死亡率
   - 30→60分钟/天：中等收益
   - 60→120分钟/天：收益递减
   - >120分钟/天：收益极小，甚至可能有害

2. **饮食优化研究**
   - 从不健康→健康：显著改善
   - 从健康→极度健康：收益有限

## 💡 如何应对严格规则

### 1. 接受现实
- 人类有生理极限
- 完美生活方式也不能永生
- 105岁已经是人类顶尖水平

### 2. 聚焦可控因素
- 不要追求不切实际的目标
- 重点改善高风险因素
- 循序渐进，持续优化

### 3. 理解递减效应
- 从0分到60分：容易
- 从60分到80分：较难
- 从80分到90分：很难
- 从90分到100分：极难

### 4. 重视关键风险
- 戒烟永远是第一优先级
- 控制饮酒至安全范围
- 保持正常体重
- 远离槟榔

---

## 🎯 优化建议

### 低收益区域（不建议过度投入）
- ACM已经低于-150%
- 继续增加积极因素收益很小
- 建议：保持现状即可

### 高收益区域（重点优化）
- ACM在-50%到+50%之间
- 改善空间大，收益显著
- 建议：优先改善负面因素

### 关键风险区域（必须处理）
- 存在重度吸烟/大量饮酒/肥胖/槟榔
- 危害巨大且持续
- 建议：不惜代价戒除

## 📊 数据透明度

### 调整公式公开

**负面因素惩罚**：
```javascript
if (negativeACM > 50) {
    penalty = (negativeACM - 50) × 0.2
    totalACM += penalty
}
```

**积极因素递减**：
```javascript
if (positiveACM < -150) {
    excess = positiveACM + 150
    diminished = excess × 0.6
    totalACM = totalACM - excess + diminished
}
```

**关键风险加权**：
```javascript
if (criticalRisks exists) {
    criticalPenalty = criticalRisksACM × 0.3
    totalACM += criticalPenalty
}
```
